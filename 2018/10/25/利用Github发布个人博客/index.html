利用Github发布个人博客

一、目的

【1】 了解github基本使用方式；
【2】 初步掌握利用HTML进行网页的设计方法；
【3】 掌握web项目中相对路径和绝对路径概念；

二、过程

【1】Github注册与配置

1.注册账号，添加库
2.下载并安装git
3.添加SSH KEY到Github
4.配置账户，测试

结果：

看到Hi xxx! You’ve successfully authenticated, but GitHub does not provide shell access. ，说明您已经成功地完成了第一步

【2】Hexo配置

1.下载并安装Node.js库
2.安装和配置hexo
    新建文件夹blog右键打开git bash,输入如下指令：
    安装： Git Bash$ npm install -g hexo-cli
    初始化：$ hexo init blog      （初始化成功后会在最后打印一行：INFO Start blogging with Hexo!）
    安装依赖文件：    $ cd blog
                    $ npm install
                    $ hexo generate
                    $ hexo server
3.修改全局配置文件_config.yml
4.安装hexo-deployer-git插件：
                    $ npm install hexo-deployer-git --save
5.部署到Github上：   $ hexo clean            清除缓存
                    $ hexo generator     生成静态页面至public目录
                    $ hexo deploy        将.deploy目录部署到GitHub

结果：

执行hexo deploy命令之后，如果最后一行打印出如下信息则表示部署成功： INFO Deploy done: git
(由于我是第二次运行hexo d 命令，且目录下并未新增文件所以显示nothing to commit)

至此，个人博客已发布成功
